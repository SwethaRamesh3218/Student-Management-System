<---------------Livewire Student Management System------------->

Description:

The Livewire Student Management System is a software application designed to streamline attendance tracking and student management for educational institutions. 
It provides features for both students and administrators to manage attendance records, view student details, and access profiles.

Features:

* Student Login: Allows students to log in using their credentials to access attendance records and other related information.
* Admin Login: Provides administrators with access to student management functionalities such as viewing attendance records and student details.
* User Authentication: Ensures secure access to the system with username and password verification.
* Intuitive Interface: Offers a user-friendly interface for easy navigation and interaction.

Installation:

* Clone the repository: git clone https://github.com/SwethaRamesh3218/livewire-student-management-system.git
* Navigate to the project directory: cd livewire-student-management-system
* Install dependencies: pip install -r requirements.txt
* Run the application: python main.py

Usage:

* Student Login:If you want to login first you login admin page and save data and then Enter your email ID and Enter your Student ID 
  to access attendance records and other student-related features.
* Admin Login: Username:livewire and Password:3218 Use the provided admin credentials to log in and access administrative functionalities.
* Navigation: Use the buttons and links provided to navigate between different sections of the application.
